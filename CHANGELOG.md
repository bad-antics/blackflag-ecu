# Changelog

All notable changes to BlackFlag ECU will be documented in this file.

## [3.0.0] - 2025-01-25

### Added
- Comprehensive vehicle database with 200+ vehicles
- Full OBD-II PID support for all standard modes
- CAN bus topology documentation
- Bench flash wiring harness diagrams
- J2534 interface configuration guide
- 45 manufacturer support

### Changed
- Simplified diagram formatting for cross-platform compatibility
- Restructured documentation for better navigation
- Enhanced connector pinout clarity
- Improved protocol detection algorithms

### Fixed
- Unicode box-drawing character display issues
- ASCII art alignment in documentation
- Cross-platform diagram rendering

## [2.5.0] - 2025-01-10

### Added
- Ford 2024 model year support
- RAM TRX ECU definitions
- Stellantis unified platform support
- Korean manufacturer (Hyundai/Kia/Genesis) expansion

### Changed
- Updated Toyota GR86/Subaru BRZ calibrations
- Enhanced Honda e:HEV hybrid support
- Improved BMW iDrive integration

### Fixed
- Nissan CVT calibration reads
- Mazda CX-90 turbo mapping
- VW MQB Evo compatibility

## [2.0.0] - 2024-11-15

### Added
- Live data streaming support
- Custom PID definitions
- Diagnostic trouble code library
- Multi-ECU scanning capability

### Changed
- Complete UI/UX redesign
- Faster ECU detection algorithms
- Reduced memory footprint

### Fixed
- J2534 device discovery on Windows 11
- USB buffer overflow handling
- Session timeout management

## [1.5.0] - 2024-09-01

### Added
- Electric vehicle battery monitoring
- Hybrid system diagnostics
- ADAS calibration support

### Changed
- Improved error handling
- Enhanced logging system
- Better diagnostic messages

## [1.0.0] - 2024-06-15

### Added
- Initial release
- Basic OBD-II scanning
- ECU identification
- DTC read/clear functionality
- 50 initial vehicle definitions
