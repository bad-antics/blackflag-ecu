# Contributing to BlackFlag ECU

Thank you for your interest in contributing to BlackFlag ECU!

## How to Contribute

### Reporting Issues

- Use GitHub Issues for bug reports and feature requests
- Include detailed reproduction steps for bugs
- Specify your Windows version and .NET version

### Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### Development Setup

1. Install Visual Studio 2022 with .NET 8.0 workload
2. Clone the repository
3. Open `BlackFlag.sln`
4. Build and run

### Code Style

- Follow C# naming conventions
- Use XML documentation for public APIs
- Keep methods focused and small
- Write unit tests for new features

### Legal

- Do not include copyrighted ECU maps/tunes
- All contributions must be original work
- By contributing, you agree to the MIT license

## Code of Conduct

Be respectful and professional. We're here to build great automotive tools together.
